# Stromation

**Workflow Replacement Studio** — Custom automation for small businesses.

Live at [stromation.com](https://www.stromation.com)

## Stack
- Static HTML/CSS/JS on GitHub Pages
- n8n for workflow automation
- Supabase for CRM
- OpenAI for AI features
- Twilio for SMS/voice
- Stripe for payments

## Structure
- `*.html` — 19 website pages
- `blog/` — 18 blog posts
- `docs/` — internal business documents
- `img/` — SVG illustrations
- `logos/` — brand assets
- `.claude/` — Claude Code configuration
