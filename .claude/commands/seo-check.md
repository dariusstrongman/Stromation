Audit SEO across all pages of the Stromation website.

Check every HTML file (root and blog/) for:
1. Missing or duplicate `<title>` tags
2. Missing or empty `<meta name="description">`
3. Missing Open Graph tags (og:title, og:description, og:image, og:url)
4. Missing Twitter card meta tags
5. Missing canonical URLs
6. Missing or incorrect structured data (JSON-LD)
7. Images missing `alt` attributes
8. Missing `<h1>` tags or multiple `<h1>` tags
9. Broken internal links (href pointing to files that don't exist)
10. Pages missing from `sitemap.xml`

Report findings grouped by issue type with the filename and line number for each problem. At the end, give a summary score (e.g., "23/25 pages pass all checks").
