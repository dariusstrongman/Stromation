Deploy the Stromation website to GitHub Pages.

Steps:
1. Run `git status` to see what has changed
2. Run `git diff` to review the changes
3. Show a summary of changes to the user and ask for confirmation before proceeding
4. Stage all changed files (but warn about any sensitive files like .env)
5. Create a commit with a descriptive message
6. Push to the main branch (which triggers GitHub Pages deployment)
7. Confirm the push was successful
