Help design or troubleshoot an n8n workflow.

Details: $ARGUMENTS

The Stromation n8n instance is at: https://n8n.stromation.com/home/workflows

Steps:
1. Understand what the user wants the workflow to do
2. Design the workflow step-by-step:
   - Trigger node (webhook, cron, form submission, etc.)
   - Processing nodes (data transformation, conditionals, loops)
   - Action nodes (send email, update CRM, create record, API call, etc.)
   - Error handling (error trigger node, notification on failure)
3. If building a new workflow, provide:
   - The node sequence with configuration details
   - Any credentials or API keys needed
   - Test data or example payloads
   - n8n JSON export if possible so it can be imported directly
4. If troubleshooting, ask for the error message or unexpected behavior and walk through debugging
